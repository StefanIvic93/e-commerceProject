SEMINARSKI RAD

Pregled seminarskog rada na temu "WebGL"

Predmet : Eletkronsko Poslovanje

"WebGL(Web Graphics Library) je JavaScript API koji se koristi za renderovanje 3D grafike unutar bilo kod web pretraživača koji ima tu mogućnost bez ikakvih plugin-ova. WebGL je u potpunosti integrisan u svim web standardima koji omogućavaju GPU u odredjenom kompjuteru ubrzano korišćenje fizike I obradu slike I efekata kao I deo web stranice. WebGL elementi mogu da se mešaju sa drugim HTML elemenata i kombinuju sa drugim delovima stranica ili pozadina stranica. WebGL programi se sastoje od kontrolnog koda koji je napisan u JavaScript i Shader-u koji je napisan u GLSL, jeziku sličan C ili C ++, a izvršava se na na grafičkim procesorima računara(GPU). WebGL je dizajniran i održava se od strane neprofitne grupe - Khronos Group."
